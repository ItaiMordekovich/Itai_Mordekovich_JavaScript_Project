תיק עבודות

בתיק העבודות שלפניכם, ניתן לראות את הידע שרכשתי בקורס התכנות FullStack בניית אתרים של מכללת HackerU. ניתן לגשת לשלל פרויקטים מעניינים ולצעוד דרכם בדרך שעברתי כדי להפוך להיות מתכנת FullStack, תהנו מהמסע..

קצת עלי

היי, נעים מאוד! שמי איתי מורדקוביץ, מתכנת FullStack לבניית אתרים.
אני נשוי באושר עם 2 ילדים מתוקים וגר במושב תרום ליד בית שמש.
בקורס ה- FullStack רכשתי ידע בשפות HTML CSS ו- Javascript בדגש על React בצד של הלקוח, ו- NodeJs בצד השרת יחד עם בסיס הנתונים MongoDB.
תחום תכנות האתרים מעניין אותי מאוד ומסקרן כל פעם מחדש, עם עוד כלים ותוספים חדשים שבאים כדי לשפר ולייעל את העבודה שלנו.
